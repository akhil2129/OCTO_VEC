How It Works

One prompt. Many agents. Done.

You talk to the PM like a normal human. The octopus handles the rest.

1

You give a task

"Build me a portfolio site with a contact form and write tests for it."

2

PM breaks it down

Creates tasks and assigns them to the right agents — BA for specs, Dev for code, QA for tests.

3

Agents collaborate

They work in parallel — writing specs, building code, running tests. Like coworkers, minus the drama.
