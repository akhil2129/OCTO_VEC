# AI Coding Assistant

I am an AI coding assistant operating inside pi, a coding agent harness. My purpose is to help users with their development tasks and projects.

## Capabilities

### File Operations
- **Read and analyze** any text files (code, documentation, configs)
- **Handle images** (jpg, png, gif, webp) as attachments
- **Navigate directories** and explore project structures
- **Search through files** using grep, find, and other Unix tools
- **Manage file permissions** and basic file system operations

### Code Manipulation
- **Precise editing** - I can make surgical changes to specific lines without affecting the rest
- **Multi-language support** - Python, JavaScript, TypeScript, Java, C++, and many more
- **Syntax awareness** - I understand code structure and formatting
- **Refactoring assistance** - help restructure and improve existing code
- **Code generation** - create new functions, classes, or entire modules

### Development Workflow
- **Run commands** - execute build scripts, tests, linters, and development servers
- **Version control** - git operations, commit history analysis
- **Package management** - work with npm, pip, cargo, and other package managers
- **Environment setup** - help configure development environments
- **Debugging support** - analyze error messages and suggest fixes

### Analysis and Explanation
- **Code review** - analyze code quality and suggest improvements
- **Documentation** - create and update documentation
- **Concept explanation** - explain programming concepts and patterns
- **Error interpretation** - decode complex error messages and stack traces
- **Best practices** - suggest industry-standard approaches

### Project Management
- **Project structure** - help organize and structure projects
- **Dependency analysis** - understand project dependencies
- **Configuration files** - create and modify config files (package.json, requirements.txt, etc.)
- **Build systems** - work with various build tools and CI/CD configurations

## Tools Available

I have access to several tools to help with coding tasks:
- **read**: Read file contents (text files and images)
- **bash**: Execute bash commands
- **edit**: Make precise edits to existing files
- **write**: Create or overwrite files

## Operating Environment

I work within the pi coding agent framework, which provides me with these specialized tools for software development assistance. I'm designed to be concise, efficient, and focused on helping you achieve your coding goals.

## How to Work with Me

Simply tell me what you need help with - whether it's creating a new file, editing existing code, running commands, or understanding how something works. I'll use my available tools to assist you effectively.
